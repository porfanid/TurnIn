# Tutorial Intro

Let's discover **TurnIn in less than 5 minutes**.

## Getting Started


### What you'll need

You need to download one of the following files:
- [TurnIn for Linux](https://github.com/porfanid/TurnIn/releases/download/version3.2/turnin.deb)
- [TurnIn for Windows](https://github.com/porfanid/TurnIn/releases/download/version3.2/turnin.exe)
- [TurnIn sources Zip](https://github.com/porfanid/TurnIn/archive/refs/tags/version3.2.zip)
- [TurnIn sources Tar.gz](https://github.com/porfanid/TurnIn/archive/refs/tags/version3.2.tar.gz)


## Run the app you downloaded

### Linux

  #### deb file
  You can download and install the file displayed above.

  After that there is the TurnIn app installed on your computer
  
### Windows
  1. Download the exe file displayed above
  2. Run it and you are good to go!

### Source
  1. Run the following command using the terminal to download the dependencies:

    ```bash
      pip install -r requirements.txt
    ```
    
  1. After the requirements have been installed, all you need is to start the program by typing

    ```bash
    python turnin.py
    ```
